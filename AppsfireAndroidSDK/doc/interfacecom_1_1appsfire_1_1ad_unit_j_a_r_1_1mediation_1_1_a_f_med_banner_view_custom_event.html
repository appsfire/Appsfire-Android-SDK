This method is called when called when the Appsfire mediation banner requires a new banner view

Parameters:
activity activity that requested the banner view
parameters this map may contain additional information provided via the custom payload entered by the publisher on the mediation dashboard. for instance, it may be used to pass information like ad placement ids
Note:
Your implementation must make sure that the size of your banner does not exceed the publishers provided size. If it does not fit you should call the listener's `didFailToLoadAdWithError` method with an error.
You will need to disable automatic ad refreshing on your banner if this one allows you to do it.
